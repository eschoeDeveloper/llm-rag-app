# ⚡ LLM-RAG-APP

---
## 📖 프로젝트 개요

**LLM-RAG-APP** 은 react 기반의 LLM+RAG 테스팅을 위한 모바일 앱으로서, OpenAI Client를 통한 LLM 및 PgVector + PostgreSQL 기반의 RAG API를 거쳐서 사용자 입력에 대한 결과를 모델링으로 받아오는 테스팅입니다.
